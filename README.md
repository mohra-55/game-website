# 🎮 Game Website

A **fun and interactive Game Website** built using **HTML5** and **CSS3**, designed to be **fully responsive** and work smoothly on desktop, tablet, and mobile devices.

---

## 🛠️ Technologies & Skills

- **HTML5 & CSS3** – structuring and styling web pages  
- **Responsive Design** – mobile-first layouts and media queries  


## 🚀 Features

- Fully **responsive design** across all devices  
- Clean and modern interface  
- Interactive elements and hover effects  
- Easy to navigate layout  
- Compatible with all modern browsers
